# Startup SaaS

Plateforme SaaS pour la gestion de projets, tâches et factures.

## Stack
- Laravel (API)
- Vue.js (Frontend)
- MySQL
- Docker

## Fonctionnalités
- Authentification
- Organisations & membres
- Projets
- Tâches + Kanban
- Facturation
- Notifications

## Lancer le projet
docker compose up -d
